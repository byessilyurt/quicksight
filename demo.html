QuickSight

AI-Powered YouTube Video Summarizer Chrome Extension

Note: This is a demo page showing the extension interface. To use QuickSight, you need to install it as a Chrome extension and use it on YouTube.com.

How to Install & Use

  1. Open Chrome and go to chrome://extensions/
  2. Enable "Developer mode" (toggle in top right)
  3. Click "Load unpacked" and select this project folder
  4. Click the QuickSight extension icon and add your OpenAI API key
  5. Go to YouTube.com and hover over video thumbnails!
